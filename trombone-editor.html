Trombone Champ Save Editor
Instructions

Mac:
1. Enter Trombone Champ and note your current number of toots/turds.
2. Go to /Users/<your-name>/Library/Application Support/com.holywowstudios.trombonechamp/
3. Copy both tchamp_savev100_0.dat and tchamp_savev100_0_BACKUP.dat somewhere safe as a backup.
4. Upload both files to the editor, choose your new numbers, and download the edited files.
5. Replace the old files in the folder with the new ones.
6. Reopen Trombone Champ and enjoy!

Windows:
1. Enter Trombone Champ and note your current number of toots/turds.
2. Navigate to C:\Users\<YourName>\AppData\LocalLow\Holy Wow Studios\Trombone Champ\
3. Backup your save files.
4. Use this editor to upload and modify your save.
5. Replace the old files with the new ones.
6. Launch the game and enjoy!
