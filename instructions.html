Trombone Champ Save Editor - Instructions

How to Use This Save Editor

Before editing your save files, you’ll need to locate them on your computer.

For macOS Users

Your save files are located in:
/Users/yourusername/Library/Application Support/com.holywowstudios.trombonechamp/

In this folder, find the files named:
tchamp_savev100_0.dat and
tchamp_savev100_0_BACKUP.dat

For Windows Users

Your save files are located in:
C:\Users\yourusername\AppData\LocalLow\Holy Wow Studios\Trombone Champ\

In this folder, find the files named:
tchamp_savev100_0.dat and
tchamp_savev100_0_BACKUP.dat

Editing Your Save Files
  1. Launch Trombone Champ and check your current number of Toots and Turds in-game.
  2. On this website’s editor page, upload both save files (tchamp_savev100_0.dat and tchamp_savev100_0_BACKUP.dat).
  3. Enter the current numbers of Toots and Turds as prompted (this lets the editor find the correct spots in your save files).
  4. Set your desired new values for Toots and Turds using the sliders or input boxes. Use the toggles to choose which values to edit.
  5. Click “Update Save” to process the files.
  6. Download the edited save files (they will have the exact same filenames).
  7. Go back to your save folder and replace the original files with these edited versions.
  8. Start the game and enjoy your updated Toots and Turds!

Important: Always keep backups of your original save files before editing, just in case!
